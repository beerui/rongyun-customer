DajiCS SDK 本地联调 Demo

模拟第三方宿主站点。SDK 通过 window.DajiCS 对外暴露。

⚠ 首次使用先跑 pnpm build:sdk && cp dist/sdk/iife/daji-cs.iife.js public/ 。

1. 基础打开（无商品卡）

只传身份 query，直接新开 /buyer/:supplierId 。

2. 预投商品卡 + 打开

先 POST /sendRyMessage （本地会失败并自动重试 2 次，预期看到 presend:error，仍然开窗）。

3. 兜底入口 / 重置

💡 标签页模式：浏览器自动复用同名窗口 'DJ_Chat_Window'，无需手动管理

4. 悬浮 Launcher（C 阶段新增）

挂载后右下角出现气泡，点击展开 iframe 浮窗（移动端全屏）。

5. 会话结束 + 身份补发（C3 阶段新增）

先挂载 Launcher 并展开 widget，再点以下按钮触发。
